Set up

We are going to use RStudio for this course. You can do this on campus from a University Computer, online using a free Posit Cloud account that you can access from any web browser, or by downloading R and RStudio onto your own device.

Click below to see the set up instructions for each:

  • University Computer
  • Online Cloud
  • Own Device

To get set up, follow the below steps. Each step has a screen recording to assist you.

NoteStep 1: Launch RStudio from AppsAnywhere
  1. Type in AppsAnywhere to the windows bar. This will open in a web browser
  2. Type in RStudio in AppsAnywhere
  3. Click Launch and wait for it to install and open.
CautionStep 2: Navigate to your OneDrive folder

  1. In the top right corner of the bottom right panel (where it says “Files”), click the three dots ... to open the file explorer

  2. Find your OneDrive, and create a new folder for this course (e.g. LS4003_R)

  3. Select this new folder and click “Open”

TipStep 3: Set your folder as the Working Directory

Once you’ve opened your folder, you need to set this as the folder we’re going to be working from.

Click the Cog and select Set As Working Directory

CautionStep 4: Create an R file

Once you’re in your folder you can create and save an R file. This is where you put your code.

  1. Click on the Green Plus icon and select “R Script”
  2. Click the Save icon
  3. Give your file a name (e.g. “R_class_1” )

This will automatically add the “.R” extension so we know it’s an R file - R_class_1.R

Warning

Make sure you can find your file in file explorer. Always back up your work such as saving in OneDrive or emailing to yourself so that you don’t lose your progress.

NoteStep 5: Write and run R code

You’re now ready to run some R code!

  1. Copy and paste the following into your R file and click save:
value <- "Hello World"
value
  1. Highlight both lines and click the “Run” icon (green arrow)

You should see a result in your console (bottom left panel) and your environment (top right panel)

You’re now ready to start learning R!
